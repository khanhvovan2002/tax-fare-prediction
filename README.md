# tax-fare-prediction
1. Performed a preliminary inspection of the data supplied by the NYC Taxi and Limousine Commission
2. Analyzed, explored, cleaned and structured prior to any modeling.
3. A/B test to analyze the relationship between credit card payment and total
fare amount
4. Predicting the estimated fare amount for taxi rides
5. Predict whether a NYC TLC taxi cab rider will be a generous tipper.
